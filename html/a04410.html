Detailed Description

template<typename GeoMap, typename Cubature>
struct gdl::intrepid::unstructured_geometry< GeoMap, Cubature >

discretization of the local-to-global map

This map is what connects the reference to the actual configuration. All the quantities and integrals computed in the reference element get eventually interpolated in the actual configuration using this map. The case in which this class uses the same basis functions as the finite element discretization is called "isoparametric".
