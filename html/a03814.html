template<typename FunctionDefinition>
struct gdl::functors::function_evaluation< FunctionDefinition >

[cubature_point_grid_generation]

[function_evaluation]

functor performing the evaluation of a mathematical function for a given point grid

Given an input grid (grid_ponts) this functor provides in output the values of the mathematical function corresponding to the grid points coordinates. The mathematical function definition is provided as template parameter.
