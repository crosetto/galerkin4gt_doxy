Detailed Description

This file contains some of the core definitions used for the discretization on the reference space.

A reference finite element is univocally determined by the basis, accessed through this file, a discretization order (a numeric template argument), and a shape (simple structure containing the topology information, we borrow most of the times the shape definitions from the Shards library in Trilinos).
