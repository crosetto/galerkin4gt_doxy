computing the operator passed as input applied to all the basis functions

This member function must be call once in a preprocessing step. It precomputes the value or gradient (depending on the input operator) of the basis functions in the cubature points of the faces.

Parameters
operator_can be Intrepid::OPERATOR_VALUE, Intrepid::OPERATOR_GRAD
orderinga vector containing an alternative order of the dofs (it is used e.g. to match the results with the DEALII library, which uses a different convention for the nodes numbering).
