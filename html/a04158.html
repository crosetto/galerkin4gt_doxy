Detailed Description

template<typename Geometry>
struct gdl::assembly< Geometry >

aggregator of tools and storages used to assemble the finite elements discretization of the most common operators

Contains an API to compute and access the elemental Jacobian, its determinant, and its inverse. The computation of these quantities can be done in the user code as well, but since they are an almost mandatory preliminary steps for all finite elements discretizations they are provided through this API.

Note
The following class holds the discretization containers spanning the whole iteration domain, as opposed to intrepid.hpp which holds the local elemental quantities.
