adjacency rule application method

This method performs the check described in struct comment and, if possible, provides updated values for the element/dof coords, corresponding to the case of a dof pair that can be seen as a single element case, assuming that coord values in the other direction are the same. In this case the true value is returned.

Firstdof coordinate in mesh element

Returns
