Detailed Description

Implementation of the finite element space local operations such as value/derivative of the basis functions

These operations are done only once, oin the host, and are local to the reference element, so their overhead is negligible.

A "reorder" macro flag can be set if we want to change the order of the d.o.f. in the local elements. This feature is only used in order to compare the results with other codes, since every code has its own convention.
