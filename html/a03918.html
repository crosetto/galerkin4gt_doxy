template<ushort_t N_DOF0, ushort_t N_DOF1, ushort_t N_DOF2>
struct gdl::functors::hexahedron_vector_distribute< N_DOF0, N_DOF1, N_DOF2 >

hexahedron mesh assebled vector (single dof indexed object) distribution functor

After the assemble operation performed by the hexahedron_vector_assemble functor, or starting from an already assembled vector, this functor performs the copy of the vector values corresponding to shared dof (with adjacent mesh elements) to the storages of the adjacent element themselves. Same dof numbering rule described for hexahedron_vector_assemble is used (see above).
