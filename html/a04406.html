Detailed Description

template<typename FE, typename Cub>
struct gdl::intrepid::discretization< FE, Cub >

defining the finite element discretization

Given the finite elements space definitions and the quadrature rule it instantiates the useful quantities (e.g. the values of the basis functions and of their derivatives) in the quadrature points.

Examples:
mass.cpp, and poisson.cpp.

computing the operator passed as input applied to all the basis functions

This member function must be call once in a preprocessing step. It precomputes the value or gradient (depending on the input operator) of the basis functions in the cubature points.
