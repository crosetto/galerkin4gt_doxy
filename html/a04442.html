Detailed Description

template<uint_t Dim, uint_t Order>
struct gdl::legendre< Dim, Order >

implementation of Legendre basis functions in 2D and 3D

Legendre orthogonal polynomial are computed from the univariate case (which is taken from the Intrepid library), by computing a tensor product space. This class only supports tensor product elements, and the evaluation of the polynomials is carried out by the helper struct evaluate_polynomial

Note
this file is messy, was developed in a quick and dirty way, but it does something simple (the tensor product of something already available), can be refactored or replaced by something nicer.
