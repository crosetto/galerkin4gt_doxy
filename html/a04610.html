creates a rectilinear grid

This grid must be fully structured hexahedral grid, but the elements can have different sizes. The grid coordinates of the edges (including the edges local to the elements) are copied into arrays.

Parameters
storage_the grid of elements
local_grid_info_a storage_info describing the local element. The underlying hypothesis is that the elements local points are ordered in a tensor-product structure.
