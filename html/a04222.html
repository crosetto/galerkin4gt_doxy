Detailed Description

template<typename Boundary>
struct gdl::bd_assembly< Boundary >

Definition of the quantities needed for performing computations on the boundary.

Contains an API to compute and access the boundary Jacobian, its determinant, the normals field.

Note
The following class holds the discretization containers spanning the whole iteration domain, as opposed to intrepid.hpp which holds the local elemental quantities. NOTE: underlying 3D assumption
