Detailed Description

template<ushort_t Order, enumtype::Basis BasisType, enumtype::Shape ShapeType>
struct gdl::reference_element< Order, BasisType, ShapeType >

class containing information about the reference element

Template Parameters
Orderthe discretization order
BasisTypethe type of basis function (Lagrange, Legendre, ...) see enumtype::Basis
ShapeTypethe type of shape (e.g. Hexa), see enumtype::Shape
Note
: So far only Hexa/Quad shapes are supported
Examples:
dg_advection.cpp, dg_flux.cpp, gather.cpp, interpolation.cpp, legendre.cpp, mass.cpp, and poisson.cpp.
