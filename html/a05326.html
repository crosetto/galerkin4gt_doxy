Interpolation between FE Spaces

This example shows how to transform and counter-transform a field using the Legendre basis. Using basis functions which are not interpolatory, as in general the Legendre polynomials, trascendental functions, b-splines, poses the problem that we, humans, live in the physical space. So when we want to assign a boundary/initial condition, or visualize a solution, we don't want to do that in terms of coefficients of a serie expansion, we want to use physical points. This introduces the need of constantly transform back and forth form the physical space to the space of the expansion coefficients. In this case we demonstrate how to transform a polynomial (an affine function) from the physical space to the space of Legendre coefficients, and back, and we check that we get back the original polynomial.

The example is structured in 3 steps:

  • first we compute the integral measure, as the determinant pof the jacobian matrix of the geometric map (using Lagrange interpolatory polynomials for the geometric map).
  • then we run the transformation, which is a volume integral, and we print the value of the coefficients
  • then we run the counter-transformation (another volume integral), and we check that we get back the original polynomial
